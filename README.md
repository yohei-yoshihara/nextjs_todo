# Next.JS 15 Todo

[こちら](https://nextjs.org/learn)を参考にして作成してみた Todo アプリ。

![スクリーンショット 2024-10-17 19 16 34](https://github.com/user-attachments/assets/d3ef546b-ed6c-4fbd-bfb2-5a3d43c80814)

## 起動手順

```
pnpm install
npx auth secret
npx prisma generate
npx prisma db push
node prisma/seed.mjs
pnpm run dev
```
